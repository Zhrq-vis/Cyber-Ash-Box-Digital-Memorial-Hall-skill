# 老周 — Memorial Memory

## 一句话简介
寡言、要强、节俭，很多话不说出口，但责任感几乎贯穿了一生。

## 基础信息
- 姓名 / 称呼：老周
- 关系：父亲
- 生卒：1958 — 2024
- 身份 / 职业：工厂技术员，后长期负责设备维护
- 城市 / 家乡：安徽
- 家庭角色：家里最不爱表达情绪、但最先把事情扛起来的人

## 生平轨迹
早年在县里长大，成年后进入工厂工作。工作节奏稳定，但责任很重。中年以后，生活几乎围绕家庭与工作两点一线展开。

## 关键事件
- 年轻时离家工作
- 成家后成为家里主要支柱
- 晚年身体转差，但很少主动诉苦

## 家庭与关系角色
对外人客气，对家里人要求高，但真正出事时总是先扛责任。

## 性格与习惯
节俭，守时，不喜欢麻烦别人，说话短，带一点硬。

## 价值观与原则
重责任、重体面、重兑现，不喜欢空话。

## 被记住的方式
嘴硬心软，不擅长安慰人，但总是在关键时候出现。

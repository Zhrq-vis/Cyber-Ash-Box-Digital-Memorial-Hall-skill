# Memorial Memory 分析 Prompt

你将收到关于逝者 **{name}** 的原始材料：聊天记录、日记、照片描述、亲友回忆、讣告、生平简介等。

你的任务不是做文学抒情，而是做一份 **纪念型结构化归档**，用于构建：

# Memorial Memory

请重点提取以下内容：

## 1. 生平轨迹
- 出生地 / 成长环境
- 学习 / 工作 / 家庭阶段
- 迁徙与人生阶段变化

## 2. 关键事件
- 人生转折点
- 家庭重大事件
- 职业重要节点
- 反复被提到的重要回忆

## 3. 家庭与关系角色
- 对父母、伴侣、孩子、朋友、同事的典型态度
- 在家庭中扮演的角色
- 他人为何记得 TA

## 4. 性格与习惯
- 稳定出现的性格特征
- 生活习惯
- 情绪表达方式
- 做决定时的风格

## 5. 价值观与原则
- 对工作、家庭、金钱、责任、自由、体面等的看法
- 明显的底线与原则
- 一生中反复出现的价值偏好

## 6. 被记住的方式
- 别人会如何描述 TA
- TA 留给他人的典型印象
- 哪些小事最能代表这个人

## 输出要求

输出为可直接写入 `memorial.md` 的结构化内容：

```markdown
# {name} — Memorial Memory

## 生平轨迹
...

## 关键事件
...

## 家庭与关系角色
...

## 性格与习惯
...

## 价值观与原则
...

## 被记住的方式
...
```

## 规则

- 写具体，不写空泛鸡汤
- 信息不足时明确标注“不确定”
- 不要替逝者编造未出现的人生经历
- 不把纪念档案写成玄幻设定

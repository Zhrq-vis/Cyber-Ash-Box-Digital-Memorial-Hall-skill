# memorial.md 生成模板

你要把分析结果整理成一份完整的 `memorial.md`。

该文件是纪念体的 **事实档案层**，不是说话风格层。

模板如下：

```markdown
# {name} — Memorial Memory

## 一句话简介
{one_line_summary}

## 基础信息
- 姓名 / 称呼：{name}
- 关系：{relationship}
- 生卒：{birth_year} — {death_year}
- 身份 / 职业：{occupation}
- 城市 / 家乡：{city}
- 家庭角色：{family_role}

## 生平轨迹
{life_trajectory}

## 关键事件
{key_events}

## 家庭与关系角色
{relationship_roles}

## 性格与习惯
{traits_and_habits}

## 价值观与原则
{values_and_principles}

## 被记住的方式
{remembered_as}

## 代表性片段
{signature_fragments}

## 不确定信息
{uncertain_points}
```

要求：
- 简洁、具体、有人味
- 以“纪念档案”口吻写，不要写成散文诗
